## Contributing

At the moment, this project is not open to external contributions, and pull requests will not be accepted.

That said, feedback is highly welcome. If you encounter a bug, have a suggestion, or would like to discuss a potential feature, please open an issue or start a discussion. Once the project stabilizes, contribution guidelines may be introduced.
